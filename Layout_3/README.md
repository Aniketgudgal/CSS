# Leadership Template — Layout 3

This repository contains the homepage created for the Leadership Conference 2025 using the "Layout 3" template. The page features a full-width header, large hero/banner, two-column content area (main + right sidebar), and a full-width footer — suitable for leadership profiles, announcements, and conference resources.
